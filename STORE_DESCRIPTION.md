## Full Description (Chrome Web Store)

**Reset or mark complete your Udemy course progress with one click.** No page reloads, no manual clicking through hundreds of lectures.

### Why this extension?

Udemy doesn't offer a built-in way to reset course progress. If you've completed a course and want to start fresh — for certification prep, deeper review, or simply a clean progress bar — you're stuck manually unchecking lectures one by one.

Udemy Reset Progress solves this in seconds.

### Features

**Clear Progress** — Resets all lecture completion checkmarks across every section of your course. One click, done.

**Mark All Complete** — Marks every lecture as completed. Useful when you've already mastered the material and want to skip to the certificate or focus on specific sections.

**Configurable Speed** — Adjust the action delay (50–500ms) to balance speed and reliability. If you experience rate-limiting, increase the delay in Settings.

**Lightweight** — Under 35 KiB. No background processes. Loads instantly.

**Privacy-first** — Zero data collection. Zero analytics. Zero external network requests. Everything happens locally in your browser. Open source: https://github.com/Shramkoweb/udemy-reset-progress

**Cross-browser** — Available for Chrome, Firefox, and Edge.

**Actively maintained** — Regular updates to keep up with Udemy's interface changes. The developer responds to every review and issue.

### How to use

1. Open any Udemy course page (the curriculum/lecture view)
2. Click the extension icon in your toolbar
3. Choose "Clear Progress" to reset or "Mark All Complete" to finish
4. Done — your course progress updates instantly

### Troubleshooting

- Make sure you're on a Udemy course page with the curriculum sidebar visible
- If progress doesn't reset, try increasing the action delay in Settings (click the gear icon)
- If you experience unexpected logouts, increase the delay to 250ms or higher

### Looking for a working Udemy progress reset tool?

Other reset extensions haven't been updated in years and no longer work with Udemy's current interface. This is the only actively maintained extension for resetting Udemy course progress — updated regularly, open source, and trusted by 1,000+ users across Chrome and Firefox.
